Python小札记(1)-对象的默认布尔值

本文主要记录在python中，不同对象在进行布尔检测时的默认值

在python中，可以将对象用于布尔检测和布尔表达式中，因此，知道哪些对象的布尔值为False，可以方便我们编写简洁的代码。

布尔值为False的对象包括下面几种：

  • SNone
  • False
  • 所有值为0的数值，包括0，0.0， 0L，0j等
  • 所有空的序列，包括字符串””, 空的元组(), 空的列表[]
  • 所有空的映射类型，例如空的字典{}
  • 对于指定的类的对象，当类中定义了nonezero（）或者len（）函数，并且其返回值为0或者False时，该对象的布尔值为False

在其他情况下，对象的布尔值都为True。
