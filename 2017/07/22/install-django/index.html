Django安装

一直想学习下web后端开发，刚好最近想把买了几个月的树莓派用起来，干脆就做一个小的视频网站吧。由于对于Python比较有好感，因此决定选择Python作为后端开发语言。为了能够快速地搞出一个能跑的版本，决定采用Django框架。

1. 安装

使用Django框架，第一步，肯定是要安装咯。目前的Django官方发行版本为1.11.3。而对于官方发行版本，安装起来也很方便，只要在命令行中使用下面的命令就可以安装了。

pip install Django==1.11.3

2. 验证

在使用上面的命令安装了之后，可以在Python交互环境中输入下面的命令进行验证

import django  
print(django.get_version())

如果上面的Python代码输出的结果为1.11,即你安装的版本的话，就表示Django已经在你的电脑上安装完成了，接下来就可以开始通过官方教程进行学习了。

参考资料

  2. https://docs.djangoproject.com/en/1.11/intro/install/
